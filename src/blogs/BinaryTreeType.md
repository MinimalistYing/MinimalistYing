# 常见的二叉树类型

## 满二叉树
除最后一层的节点外每个节点都有俩个子节点
```
           1
         /   \
        3     2
       / \   / \ 
      5   7 8   9 
```

## 完全二叉树
除最后一层外每层节点全满，且最后一层节点全部靠左排列  
满二叉树一定是完全二叉树
```
           1
         /   \
        3     2
       / \   / 
      5   7 8 
```

## 平衡二叉树 （AVL Tree）
空树或者左右子树高度最多相差一层且左右子树都为平衡二叉树
```
           1
         /   \
        3     2
       / \   
      5   7 
```
