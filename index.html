正则

2015年11月开始实习，开始了漫长的职场生涯，至今共参与了俩款产品的研发工作。
第一个项目是与阿里云合作进行公安部门大数据平台的建设。
jQuery、Bootstrap、require.js、Spring MVC、Hibernate、Maven、SVN。
刚开始主要从事应用自身所需的各类标签数据管理页面以及相应的后台数据库操作开发。
在熟悉项目后开始独立开发积分模型模块，通过对已有标签设置不同分值再将符合一定阀值及时间段的目标筛选并在页面上展示。
在其中实现过数据以CSV格式导入导出，对筛选出的大数据运用ECharts进行可视化展示，通过具体规则进行数据二次筛选等功能。
后台方面实现过类似阿里云提供的分析引擎，将前台组装的查询JSON直接转为查询数据所需的Sql语句，在大数据量情况下的多线程导入数据功能等。
除了编程所需的技能外还熟悉Maven SVN Git 等项目管理以及版本控制工具。
后转为前端，对前端模块化、工程化略有心得。喜欢通过不断优化工作流程来提高效率、减少错误。
熟悉Gulp，也尝试过一下Webpack，觉得这些工具的出现是前端工程师的福音。
